## 這週學了一大堆以前搞不懂的東西，你有變得更懂了嗎？請寫下你的心得。

這週學習的東西，有種打通任度二脈的感覺，以前看過的很多專有名詞，這一週都介紹到了，也透過作業的方式，一步步寫下底層是如何運作，反而可以更加深印象，其實概念的東西還蠻容易忘記的，看完影片的當下，你以為你懂了，其實並不。

但上完課後，最不熟悉的還是物件導向，還有作業四寫的超心虛的，感覺就是大概知道怎麼用，但好像也沒有完全懂他的運作，就是蠻模糊的感覺。

還有在看完本週檢討，才發現原來自己在  Web API 上理解也錯誤，像是 setTimeout 他只是剛好也是屬於  Web API，而且  Web API 並非一個特別的程序，只是當我們將 setTimeout 的 function 放入 call stack 裡去執行，setTimeout 會去與瀏覽器溝通，請瀏覽器設定 Timer ，完成後再將 setTimeout 的 cb function 整個放到 callback queue 等待 call stack 全部任務結束後，才依序執行 callback queue 內的 cb function。
